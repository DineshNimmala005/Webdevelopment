AMD Radeon RX 5700 XT The Radeon RX 5700 XT is a high-end graphics card by AMD, launched on July 7th, 2019. Built on the 7 nm process, and based on the Navi 10 graphics processor, in its Navi 10 XT variant, the card supports DirectX 12. This ensures that all modern games will run on Radeon RX 5700 XT. The Navi 10 graphics processor is an average sized chip with a die area of 251 mmÂ² and 10,300 million transistors. It features 2560 shading units, 160 texture mapping units, and 64 ROPs. AMD has paired 8 GB GDDR6 memory with the Radeon RX 5700 XT, which are connected using a 256-bit memory interface. The GPU is operating at a frequency of 1605 MHz, which can be boosted up to 1905 MHz, memory is running at 1750 MHz (14 Gbps effective).
